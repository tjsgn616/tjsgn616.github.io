Python - 글자 색 변환

  • 출력할 Text에 색을 입히는 방법을 기록한다.

글자 색 변환

  • 기본 형태

  • ‘\033[색깔코드m’은 뒤의 글자를 특정 색깔로 출력하게 하는 명령어 이다.

  • ‘\033[0m’은 색을 원래대로 되돌리게 하는 명령어 이다.

  • 예시

  • ‘ ‘ 사이의 텍스트는 굳이 ,를 붙일 필요 없고, 사이에 함수식이 들어가는 경우에는 ,, 로 분류를 해주어야한다.

  • 글자 색상 표

배경 색 변환

  • 글자 색 변환과 같으며 글자와 배경색을 동시에 변경 할 시, ‘ ‘ 사이에 글자색,배경색 변경 코드를 함께 입력한다.
  • 글자 색, 배경 색 동시 변환 예제

RGB 변환

  • 좀 더 다양한 색상을 이용하고 싶을 때, RGB 값을 입력하여 변환 할 수도 있다.

  • 색상 코드를 입력하는 자리에 RGB값을 입력한다.
