VSCode - 웹 개발을 위한 설정

  • VS Code가 이미 설치 되어있다는 전제 하에 진행한다.
  • VS Code에 웹개발을 위한 설정을 진행한다.
  • 확장(ctrl+shift+x) 탭 에서 진행한다.

한글 설정

  • VSC에서 지원하는 한글 모드를 사용 할 수 있다.

  1. Korean 을 검색하여 Korean(사용법) Language를 설치한다.

  2. 설치 후 VSC프로그램을 재 실행한다.

Beautify 설정

  1. 확장 탭에서 코드 자동 정렬 기능인 Beautify를 설치한다.

  2. 기능 기여도 란에 HookyQR.beautify 를 복사한다.

  3. 상단 메뉴의 코드→ 기본설정→ 바로가기 키 를 누른다.

    • Ctrl + Shift + P 로 나오는 검색창에 바로가기 를 입력하여 ‘기본 설정: 바로 가기 키 열기’ 를 눌러도 된다.

  4. 복사했던 HookyQR.beautify 를 붙여넣는다.

  5. Beautify Selection 을 선택한다.

  6. 기능을 사용할 버튼을 입력한다.

    • Ctrl + Alt + L 로 설정하였다.

  7. 코드 입력 후 Ctrl + Alt + L 입력 시 자동 정렬 된다.

태그 이름 변경

  • 태그 이름을 한번에 변경할 수 있는 플러그인을 설치한다.

  • 확장 탭에서 Auto Rename Tag 를 설치한다.

자동 업데이트 툴

  • 변경사항을 자동으로 업데이트 해주는 플러그인을 설치한다.

  • 확장 탭에서 Live Server를 설치한다.
