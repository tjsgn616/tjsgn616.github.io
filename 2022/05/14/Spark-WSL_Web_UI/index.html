Spark - WSL을 이용한 Web UI만들기

  • Spark Tutorial용 파일로 Web UI를 구동해 본다.
  • 환경 변수 설정은 끝난 상태에서 진행한다.

  1. 원하는 위치에서 가상 환경을 생성하고 접속한다.

    • temp 폴더에서 진행하였다.

    virtualenv venv

    source venv/bin/activate

  2. PySpark를 설치한다.

    pip install pyspark

  3. 데이터를 생성한다.

    • 데이터 폴더를 만들고 안에 임의의 데이터 파일을 만든다.

      mkdir data

      cd data

      vi README.md

    • READEME.md 파일 내용으로 임의의 영어 텍스트를 입력한다. (현 테스트에서는 아래의 텍스트를 이용하였다.)

  4. SimpleApp.py를 작성한다.

    • 다음과 같은 파일을 작성한다.
      • README.md 파일을 불러와서 a와 b 의 갯수를 세는 명령문이다.
      • 중간에 들어간 input() 명령어는 사용자가 추가 입력을 하지 않으면 spark 세션이 열린 상태로 유지하도록 하게하는 Trick이다. - 유용하다
    1. Spark-Submit

    • bin/spark-submit script 라고 하기도 한다.

      • 참고: https://spark.apache.org/docs/latest/submitting-applications.html

      $SPARK_HOME/bin/spark-submit --master local[4] /mnt/c/temp/SimpleApp.py

      • 맨 뒤에는 SimpleApp.py가 있는 자신의 경로를 적는다.

    1. Web UI 확인
