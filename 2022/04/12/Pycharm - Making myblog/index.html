Pycharm - myblog 만들기

2022-04-12

#Pycharm, myblog

myblog 만들기

  • Pycharm을 이용해 Github와 연동된 블로그를 만드는 과정이다.

필수 파일 설치

  • node.js 와 git을 설치
    • 링크: nodejs.org / git-scm.com
    • 설치 후 정상적으로 설치 되었는지 확인해본다.
    • node -v
    • git —version
  • git bash에서 hexo 설치
    • npm install -g hexo-cli

블로그 폴더 만들기

  1. 바탕화면에 우클릭, Git Bash 선택
  2. hexo init myblog 입력 = myblog 폴더가 생성됨
  3. myblog - 우클릭 - Pycharm으로 실행
  4. 하단부 terminal을 열고 Git Bash 란을 열어놓음.

GitHub 설정

  1. git hub에서 자기 프로필 repositories에서 새로 폴더 만들기 (myblog) 진행
  2. 아이디/myblog를 만들면 나오는 명령어들 Pycharm 터미널 Gitbash 에서 실행시킴

Myblog 폴더 설정

  • _config.yml 파일 선택

    • URL 정보 수정

    • 맨 아래로 내려서 수정

배포용 Github 설정

  • 깃허브에 “아이디.github.io” 로 repo를 하나 더 만든다.
  • 그 후 다시 Pycharm GitBash Terminal에서 명령어로 헥소를 이용해 테스트한다.
    • (localhost : 4000) 들어가서 확인

최종 배포

  • 최종 배포 또는 수정 후 재배포 시

hexo generate hexo deploy

새로운 파일/내용 추가

  • 헥소 기본적인 코딩 문법들

링크: https://hexo.io/ko/docs/commands.html
