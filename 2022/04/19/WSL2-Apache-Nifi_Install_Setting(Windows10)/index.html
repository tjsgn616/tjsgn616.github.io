WSL2 - Apache Nifi 설치와 설정(Windows10)

WSL2 환경에서 Apache Nifi를 설치하고 환경을 세팅하는 방법을 기록하였다.

JAVA 및 Apache-Nifi 설치

  1. Wsl2 에서 JAVA 를 설치한다.

    sudo apt-get update && sudo apt-get upgrade

    sudo apt install openjdk-11-jre-headless

    • 환경변수를 설정한다.

      vi ~/.bash_profile

      export JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64

      source ~/.bash_profile

  2. curl을 이용해서 NiFi를 현재 경로에 내려받는다.

    sudo wget https://downloads.apache.org/nifi/1.16.0/nifi-1.16.0-bin.tar.gz

    → 여기서 Permission Error가 발생한다면, wsl2를 관리자 권한으로 재실행한다.

  3. 다운로드 받은 tar.gz 파일의 압축을 푼다.

    sudo tar xvzf nifi-1.16.0-bin.tar.gz

Nifi 환경 설정

  1. 압축파일을 푼 다음에 nifi-1.16.0 폴더에 접속한다.

    cd nifi-1.16.0/bin

  2. ls 명령어로 nifi-env.sh 파일이 있는지 확인하고 vi 에디터로 연 후, .bash_profile에서 한 것과 같이 자바 환경변수를 잡아준다.( export JAVA_HOME=”~~~~” )

    sudo vi nifi-env.sh

    export JAVA_HOME="/usr/lib/jvm/java-11-openjdk-amd64"

  3. nifi.sh 파일을 실행한다

    sudo ./nifi.sh start

    • 결과

    Java home: /usr/lib/jvm/java-11-openjdk-amd64

    NiFi home: /nifi-1.16.0

    Bootstrap Config File: /nifi-1.16.0/conf/bootstrap.conf

  4. Webserver 주소를 확인한다.

    /nifi-1.16.0/conf$ cd .. (경로를 nifi-1.16.0/conf$ 상태로 만든 후)

    /nifi-1.16.0/conf$ vi nifi.properties

    nifi.web.https.host= 와 nifi.web.https.post= 을 확인한다.

  5. Nifi 화면에 접속한다.( https://127.0.0.1:8443/nifi/login)

    • 로그인 화면이 나온다.
    • Username과 Password를 기본 설정에 추가해 주어야한다.
    • sudo ./bin/nifi.sh set-single-user-credentials human 1234567890123
    • 여기서는 username: human , password: 1234567890123로 진행한 것이다.

  6. 로그인 후, nifi 를 재실행한다.

    sudo ./bin/nifi.sh start

    • 정상적으로 user name과 password가 입력됐다면, 접속이 완료된 것을 확인할 수 있다.
