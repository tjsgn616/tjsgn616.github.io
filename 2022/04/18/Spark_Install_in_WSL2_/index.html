WSL2 - Spark 설치(Windows)

2022-04-18

#WSL2, Windows, Spark

Spark Install in WSL2

  • PySpark를 WSL2에서 설치하는 과정

JAVA 및 Spark 설치(이미 설치 돼 있을 시 생략가능)

  • 자바 및 Spark 파일 설치

    sudo apt-get install openjdk-11-jdk

    sudo wget https://archive.apache.org/dist/spark/spark-3.2.0/spark-3.2.0-bin-hadoop3.2.tgz

    sudo tar -xvzf spark-3.2.0-bin-hadoop3.2.tgz

.bashrc 파일 수정(환경 변수 설정)

  • vi ~/.bashrc 파일에 들어가서 코드를 작성한다.
    • JAVA_HOME 과 SPARK_HOME 은 자신에게 맞는 경로로 기입한다.
    • pwd = 자신이 있는 경로 알기

테스트

  • pyspark를 실행해본다.

    • spark가 있는 폴더로 가서 pyspark 입력

    • rd = sc.textFile(”README.md”)

    • rd.count()
