WSL(Linux) - 기본 명령어/vi 명령어

파일 및 디렉토리 명령어

  • clear : 콘솔에 적힌 텍스트 전부 삭제
  • pwd : 현재 작업 중인 디렉토리를 보여줌
  • ls : 현재 위치의 디렉토리 목록
  • cd : 디렉토리 변경, cd .. : 상위 디렉토리로 이동
  • mkdir 폴더이름 : 폴더 이름 폴더를 만든다.
  • touch 파일이름.확장자 : 파일이름.확장자 파일을 만든다.
  • mv A B : 파일(이름) 바꾸기(A파일을 B파일로 이름을 바꾼다)
    • mv A 경로 : A를 해당경로로 이동
    • mv A B 경로: A B(추가로 더 가능) 파일을 경로로 이동
  • rm 파일이름 : 파일이름 삭제
    • rm -rf 폴더 : 폴더 전체 삭제
  • cp A B : A를 B라는 이름으로 복사
    • cp -f A B : B이름이 이미 있다면 삭제하고 복사
    • cp -r A B : 폴더 복사(하위내용 포함)

Vi 명령어

  • vi 파일이름.확장자 : 파일 내용을 보고 수정할 수 있다.
    • vi 내부 명령어
      • 명령모드(기본상태)
        • x : 커서가 위치한 글자 삭제(10x : 커서 있는 곳 부터 10개 삭제)
        • dw : 커서가 있는 곳 부터 띄어쓰기가 있는 곳 까지의 한 단어 삭제
        • dd : 커서가 위치한 한 줄 삭제
        • u : 실행 취소(ctrl + z)
        • yy : 복사 (5yy: 5줄 복사)
        • dd : 잘라내기 (5dd: 5줄 잘라내기)
        • p: 붙여넣기
        • 0 : 줄의 맨 처음으로(Home)
        • $ : 줄의 맨 뒤로(End)
        • G(대문자 G) : 파일 마지막 위치로 이동
        • insert / i : 입력 모드 전환
      • esc : 명령 모드 전환
      • : = 마지막 줄 모드 전환
        • :w : 저장
        • :w 파일명 : 파일명 으로 저장
        • :q = 나가기, q! = 저장하지 않고 그냥 종료
          • :wq 저장 후 종료
        • /문자열 : 커서 위치 앞 쪽의 문자열 탐색
        • ?문자열 : 커서위치 뒤 쪽의 문자열 탐색

압축 명령어

  • tar : 압축 관련 명령어
    • tar -cvf A.tar B : B를 A.tar 로 압축한다.
    • tar -zcvf A.tar.gz B : B를 A.tar.gz 로 압축한다.
    • tar -xvf A.tar : A.tar 파일의 압축을 푼다.
    • tar -zxvf A.tar.gz : A.tar.gz 파일의 압축을푼다.

패키지 설치/삭제 명령어

  • sudo : 관리자 권한 부여
  • apt-get install/uninstall 패키지이름 : 패키지 설치/삭제(주로 sudo와 함께 사용된다.)
  • apt-get upgrade : 패키지들을 최신 버전으로 업그레이드 할 때 사용
