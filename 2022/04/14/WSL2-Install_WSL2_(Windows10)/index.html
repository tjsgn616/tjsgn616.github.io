#WSL2, Windows10

WSL2 - Install WSL2 (Windows10)

  • Windows 환경에서 리눅스를 사용할 수 있게하는 WSL2를 설치한다.

1. 윈도우 버전 확인

  • Windows 10 20H1 이상의 버전에서만 설치 가능

    • 만약 버전이 낮다면 Windows Update 설정에서 최신 버전으로 업데이트를 진행한다.

2. DISM으로 WSL 기능 활성화

  • 윈도우즈 터미널을 관리자 권한으로 실행한다.(cmd 또는 terminal 검색)

  • DISM(배포 이미지 서비스 및 관리) 명령어로 Microsoft-Windows-Subsystem-Linux 기능을 활성화한다.

    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

  • DISM 명령어로 VirtualMachinePlatform 기능을 활성화한다.

    dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

  • 재부팅 한다.

3. WSL2 Lunux 커널 업데이트

  • WSL_update_x64.msi 파일 설치를 진행한다

    • 링크: https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi

  • 설치가 완료되면 윈도우 터미널을 열고 기본 WSL 버전을 WSL2버전으로 바꿔준다. (관리자 권한 잊지 말 것)

    wsl --set-default-version 2

4. Microsoft Store 에서 리눅스 설치

  • Microsoft Store 앱 에서 Ubuntu를 검색하여 설치한다.

    • Ubuntu는 가장 많이 사용되는 리눅스 배포판이다.
    • 설치가 끝나면 터미널이 열리고 자동으로 설치를 진행하므로 종료하지 않는다. 만약 터미널이 열리지 않는다면 Ubuntu를 직접 실행해 준다.
    • ’Installing. this may take few minutes…’ 메시지 후, Ubuntu에서 사용할 Username과 Password를 설정 해준다.

  • 터미널을 실행하여 wsl 버전을 확인해 본다.

    • wsl -l -v

      • 2버전이라면 그대로 사용, 1버전이라면 아래 명령을 실행한다.

        wsl --set-version Ubuntu 2

  • 참고: https://www.lainyzine.com/ko/article/how-to-install-wsl2-and-use-linux-on-windows-10/
