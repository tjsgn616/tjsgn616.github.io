WSL2 - VSCode 설치 및 연동

  • WSL에서 VSCode를 바로 실행시킬 수 있도록 연동 시킨다.

VSCode 설치

  • 링크: https://code.visualstudio.com/download
    • 관리자 실행을 위해 System Installer로 다운받는다.
    • 설치 파일을 관리자 권한으로 실행한다.
    • 설치 진행 시 환경 변수(PATH에 추가)란만 체크돼있는지 확인한다.
    • 설치 완료 후, 재부팅을 한다.

Remote WSL 연동

  1. VSCode를 실행한다.

  2. Extension 버튼을 들어가서 Remote WSL을 설치한다.

    • 체크 박스 둘 모두 체크하고 Mark Done을 실행한다.

  3. 새로운 파일을 실행한 다음 터미널에서 WSL이 있는지 확인한다.

WSL에서 VSCode 실행

  • WSL에서 최초로 Code . 입력 시 자동으로 설치

  • 설치 완료 후 Code . 입력시 해당 디렉토리로 VSCode가 실행된다.
